# SC単語帳

情報処理安全確保支援士（SC）試験の学習用フラッシュカード（PWA）。

- **覚え方**：略称 → 英語フル → 単語ごとの直訳 → 直訳の合成 → 機能 → 試験での問われ方
- 英語フルは端末の読み上げ機能（Web Speech API）で発音
- 復習は間隔反復（Leitner方式）。進捗は端末内に保存され、外部送信はありません
- インストール不要・オフライン対応（ブラウザの「ホーム画面に追加」でアプリ化）

## 使い方

ブラウザでページを開き、「毎朝の復習をはじめる」から学習。スマホでは「ホーム画面に追加」推奨。

## カードの追加・編集

`cards.js` の `CARDS` 配列にオブジェクトを足すだけ。

```js
{
  id: "一意のID", cat: "カテゴリ", abbr: "略称",
  full: "英語フル",
  words: [{ en: "単語", ja: "直訳" }],
  literal: "直訳の合成",
  func: "機能の説明",
  exam: ["試験での問われ方1", "問われ方2"]
}
```

## 技術

ビルド不要のバニラ HTML / CSS / JavaScript。外部ライブラリ・解析タグ・外部通信なし。
